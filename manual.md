# PJ LinChan User’s Manual
Control Projector via PJlink
PJLinkの仕組みを使ってプロジェクターを制御するアプリケーションです。

## 各部の説明と機能
### 管理ブロック
#### 操作ロック
- サイドパネル全体のロック、アンロックを切り替え
#### IPアドレス欄、追加ボタン
- IPアドレスを入力
  - サブネットは自動入力
- [ 追加 ] で IPアドレス欄に入力したプロジェクターを追加<br>
  - [ enter ] キーでも確定化<br>
  - [ esc ] キーが押されるまで連続で入力可
#### 探査ボタン
- 同一サブネットのプロジェクターを自動追加
#### 更新ボタン
- メイン窓のプロジェクターの状態を更新
#### ステータスエリア
- 処理の内容や終了をお知らせ
### 一括ブロック
#### Power
- 電源の入り
- 電源の切り
#### Mute ON / Mute OFF
- ミュート（シャッター）のオン / オフ
#### Input Select
- 入力ポートの切り替え
- （メーカーによって表記が違うので当てにならない）
### 整列ブロック
#### ソート
メイン窓のプロジェクターの並び替え
- IPアドレス順（自動）
- 名前順（自動）
- リスト編集（手動）
#### 特別ブロック
データ管理
- 状態保存
- 初期化

テーマ切り替え
- ライト
- ダーク
### メインブロック
#### Target
- Preset（後述）登録時の有効無効選択
#### プロジェクターアイコン
状態表示
- オフライン
- 電源オフ
- 電源オン、ミュートオン
- 電源オン、ミュートオフ

アイコンクリック
- ミュートオン、オフ

アイコン右クリック
- Webアクセス
- 電源オン
- 電源オフ
- 名前の変更
- 削除

名前クリック
- 入力切替
### プリセットブロック
クリック
- 実行（呼び出し）

[ shift ] + クリック
- 登録（状態保存）
  - プロジェクターアイコン上の Target 連動

右クリック
- 名前の変更
- 登録を解除
### キーボードアサイン
- 矢印キー ↑ 上	：全台ミュートオフ

- 矢印キー ↓ 下	：全台ミュートオン

- [ m ] 			：全台ミュートオン / オフ（トグル動作）

- 数字キー [ 1～0 ]	：Preset 1 ～ 10 呼び出し

## その他の機能
### UDP / TCP 被コントロール
外部アプリケーションなどからUDP または TCP でコントロール

ポート
- 5000 ( UDP )、5001 ( TCP )

呼び出し
- Preset：[ preset 1 ] または [ p1 ]
- Mute：[ mute on ] [ mute off ]
- Power：[ all power on ] [ all power off ]
